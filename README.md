I interpreted the problem statement as building a wireframe UI for a review intelligence solution that helps ecommerce brand owners manage and extract actionable insights from thousands of reviews across multiple platforms.

Based on that, I designed a dashboard focused on:

* review volume and sentiment monitoring
* platform-wise visibility
* AI-generated review summaries
* recurring issue detection
* actionable insights and response workflow
